Greibach Normal Form (GNF) requires every production to be of the form A → a α, where a is a terminal and α is zero or more non-terminals. Supports multi-character symbols and quoted terminals.

📝 Grammar Input

Separate symbols with spaces. Use '+' for quoted terminals, ε for epsilon. LHS symbols are non-terminals; other symbols become terminals. Use -> between LHS and RHS.
